## 修复卡牌消除位置问题（17:15→17:20）
- 用户反馈：三张卡牌消除时，会在卡槽区中心点位置重合然后消失（第二次反馈：看不出重合动画）

  2. ✅ 修改`matchPop`动画：

     - 30-60%：移动到中心点（translate到--tx, --ty）
     - 60-100%：在中心点缩小消失（scale 1.1 → 0, opacity 1 → 0）
  3. ✅ 调整动画时间从300ms到400ms，给移动效果更多时间
- 效果：三张卡牌会从各自位置移动到卡槽区中心点重合，然后一起消失，配合碎冰颗粒特效
